# **App Name**: Mapa de Diversidade

## Core Features:

- Community Grid: Display a grid view of registered communities and organizations with search, pagination, total record count, and action buttons (view, edit, archive).
- CRUD Operations: Implement CRUD (Create, Read, Update, Delete/Archive) functionality for managing community data, mirroring the provided table schema.
- Interactive Map: Integrate Leaflet to visualize communities on a map using their latitude and longitude coordinates.

## Style Guidelines:

- Primary color: Dark Green (#2E8B57). This represents growth, stability, and the environment, aligning with the values of these communities.
- Background color: Very light green (#F0FAF4), a subtle and desaturated variation of the primary color to maintain a natural feel while providing a light theme.
- Accent color: Yellow-Orange (#D4A22B). Used for interactive elements, drawing inspiration from the warmth of the sun and highlighting important actions.
- Headline font: 'Belleza' (sans-serif), bringing personality and aligning to art and design. Body font: 'Alegreya' (serif) bringing an elegant, intellectual, contemporary feel, suitable for longer texts.
- Use icons representing community, land, and culture. These will enhance usability and visual appeal within the map and data grids.
- Employ a clean, grid-based layout for both the data tables and the map interface. Ensure the application is responsive across various screen sizes.
- Implement subtle transitions for data loading and map interactions to improve user engagement without overwhelming the interface.
